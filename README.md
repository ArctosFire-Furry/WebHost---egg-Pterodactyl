# 🥚 WebHost---egg-Pterodactyl

Bienvenue sur ce dépôt ! Ici, tu trouveras des configurations (eggs) personnalisées pour le panel **Pterodactyl**, optimisées pour l'hébergement web et les services associés. ✨

## 🐾 À quoi ça sert ?
Ce projet permet d'importer facilement des environnements pré-configurés dans ton instance Pterodactyl. Que ce soit pour du PHP, du HTML/CSS ou des bases de données, ces "œufs" sont conçus pour être stables et prêts à l'emploi.

## 🚀 Installation
1. Télécharge le fichier `.json` de l'œuf qui t'intéresse dans ce dépôt.
2. Rends-toi sur ton **Panel Admin Pterodactyl**.
3. Va dans la section **Nests** puis clique sur **Import Egg**.
4. Sélectionne le fichier JSON et associe-le au Nest de ton choix (ex: WebServices).
5. Et voilà ! Ton nouveau serveur est prêt à être déployé. 🐾

* **Configurations optimisées** pour le développement web.
